Aura.js Features

Aura.js - the 12KB ultra-lightweight animation library with zero dependencies. Pure vanilla JS magic for butter-smooth 60 FPS on any device.

Current Core Features

These are available right now - all chainable, GPU-accelerated where possible.

Glow & Float

- Neon glow and gentle floating motion

Orbit & Parallax

- 3D orbital paths and scroll depth

Scroll Triggers

- Animate on scroll viewport

Auto-Hover Effects

- Automatic animations on enter

Gyro & Device Control

- Tilt-to-move and gyro parallax using native DeviceOrientation API

Voice Command

- Speak to trigger animation with built-in Web Speech API

Mood Detection

- Basic emotion-based triggers via lightweight webcam pattern analysis

Soul Detection

- Anti-bot human verification through subtle interaction pattern

E-commerce Automation

- Fly-to-cart, price pulse, auto-hover for product cards

Built-in WebGL/WebGPU Support

- Native GPU acceleration support for advanced 3D and effects

Timeline System

- Sequence animations with delays and overlays ( timeline().add(effect).play() )

Presets System

- Ready-made animation combos ( presets('heroEntrance') )

All chainable:
aura('hero').glow('80').gyroControl().voiceCommand().timeline().add('.card','float').play();

Future Additions (Tiny & Manageable - Total < 15 KB Added )

These 15 planned features will stay under 15 KB gzipped combined, using only native browser APIs - no external dependencies, no bloat.

Stagger Animation

- Sequential delay across multiple elements (stagger(100))

Loop Control

- Infinite or finite repeats (loops(true) or loop(5))

Easing Presets

- Built-in curves like bounces, elastic, easeInOut (ease('bounce'))

Keyframe Builder

- Multi-step custom animations (Keyframes({0:scale (1), 50: rotate(180) }))

Hover State

- Dedicated hover trigger (onHover(in,out))

Click Trigger

- Animate on click (onClick(effect))

Viewport Once

- Trigger only once when in view (enterOnce())

Random Variation

- Subtle randomness for organic feel (random(50))

Color Pulse

- Smooth color transitions (pulseColor('#00ffea'))

Text Reveal

- Typewriter or letter-by-letter reveal (revealText('slow'))

Particle Burst

- Lightweight canvas burst on trigger (burst(30))

SVG Path Morph

- Simple path animation (morph(toPath))

Spring Motion

- Natural bouncy physics (spring(tension:200))

Lazy Load

- Auto-initialize on scroll (lazy())

Cleanup Method

- Remove animations cleanly (destroy())
