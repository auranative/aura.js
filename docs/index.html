Aura Documentation

Aura.js is an ultra-lightweight (12KB core), dependency-free Javascript library designed to merge 2D, 3D,animation and interaction into single unified system. It enables developers to create immersive and dynamic web experiences while miantaining high performance, simplicity and future-readiness.

Aura.js allows developers to render 2D and 3D content inside the same element at the same time, eliminating the need for separate canvases or multiple frameworks.It supports WebGL and WebGPU natively, ensuring compatibilty with current and next generation rendering technologies.

The library integrates modern interaction layers including gyro control, voice control, hover triggers and parallax motion, making it well-suited for interactive interfaces, portfolios, gamified experiences and E-commerce automation. Aura.js includes visual effects such as glow, float, orbit and mood based color adaptation to enhance storytelling and UI motion.

Built for efficiency, Aura.js guarantees 60 FPS performance across devices, from desktop to low-end mobile phones, using optimized rendering pipelines and minimal overhead. It embraces future-proof standards, supporting WebGPU, modular expansion and scalable architecture, while remaining simple enough for developers who want to avoid bloated ecosystems.

Aura.js is fully open-source under the MIT license, allowing personal and commercial use without restrictions. It is positioned as an alternative to combining multiples libraries like Three.js, GSAP, Locomotive Scroll and WebGL helpers - offering a unified,modern and accessible tool that does not force complexity or dependencies
